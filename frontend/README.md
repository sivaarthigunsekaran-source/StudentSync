# StudentSync Frontend

This project is a React + Vite frontend for the StudentSync application.

## Project Structure

- `public/` - static assets for the app
- `src/` - React source files
- `src/components/` - reusable UI components
- `src/pages/` - page-level views and route targets
- `src/services/` - API integration logic
- `index.html` - app shell entry point
- `package.json` - dependency and script configuration
- `vite.config.js` - Vite build configuration

## Scripts

- `npm install` - install dependencies
- `npm run dev` - start development server
- `npm run build` - build production assets
- `npm run preview` - preview the production build

## Notes

This app uses Bootstrap 5 and React Router DOM for navigation.
