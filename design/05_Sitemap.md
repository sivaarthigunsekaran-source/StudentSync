# Navigation Rules

- Landing Page is accessible to everyone.
- Login is required to access student features.
- Sidebar is available after successful login.
- Dashboard acts as the central navigation page.
- Every module can be accessed from the sidebar.
- Profile menu is available from the top navigation bar.
- Logout returns the user to the Login page.
