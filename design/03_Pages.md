# Example

## Dashboard Page

### Purpose

Displays all important student information after login.

### Access

Student

### Components

- Sidebar
- Top Navigation
- Welcome Card
- Life Score Card
- Today's Schedule
- Upcoming Assignments
- Upcoming Exams
- Attendance Card
- Productivity Chart
- Notifications
- Quick Actions

### Buttons

- Add Task
- Start Study Session
- View Calendar
- Update Profile

### Navigation

Login → Dashboard

Dashboard → All Modules

---

## Study Planner

### Purpose

Helps students plan and monitor study progress.

### Components

- Subject List
- Topic List
- Progress Bar
- Study Timer
- Revision Schedule

### Buttons

- Add Subject
- Add Topic
- Start Study
- Complete Topic

---

## Assignment Tracker

### Purpose

Manage assignments and submissions.

### Components

- Assignment List
- Due Dates
- Priority
- Status
- Attachments

### Buttons

- Add Assignment
- Edit
- Delete
- Mark Complete

---

## Life Score Dashboard

### Purpose

Displays the student's overall performance.

### Components

- Study Score
- Health Score
- Sleep Score
- Mental Wellness Score
- Finance Score
- Placement Score
- Productivity Score
- Overall Life Score

### Charts

- Weekly Progress
- Monthly Progress
- Trend Analysis

# Navigation Rules

- Every page should have a Back option where required.
- Dashboard should always be accessible.
- Sidebar remains visible after login.
- Profile menu available in the top-right corner.
